# Notes

This folder now contains a standalone Next.js study site built for deep learning and future expansion.

## What is included

- A landing page for the study roadmap
- A design-patterns section with original notes and diagrams
- A high-level design section with a senior-level locking/contention note
- Reusable content primitives for paragraphs, highlight blocks, tables, and SVG diagrams
- Static-export friendly routing so the site can be hosted as static files later

## Content policy

The design-pattern notes are original summaries and explanations inspired by the subject, not copied text.
That keeps the project safe to publish and easier to extend with your own voice.

## Run locally

If you already have the shared `node_modules` from the other workspace, you can run:

```bash
npm run dev
```

The app uses port `3001` so it can run alongside your other Next project if needed.
